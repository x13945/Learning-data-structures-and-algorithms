# 排序算法

排序算法对比

| 算法 | 是否是原地排序 | 是否稳定 | 最好时间复杂度 | 最坏  | 平均  |
| ---- | -------------- | -------- | -------------- | ----- | ----- |
| 冒泡 | 是             | 是       | O(n)           | O(n²) | O(n²) |
| 插入 | 是             | 是       | O(n)           | O(n²) | O(n²) |
| 选择 | 是             | 否       | O(n²)          | O(n²) | O(n²) |
